🧾 RUNBOOK - CARGA DE INTERNACIONES EN CIPRES

Fecha: 04/08/2025 · Autor: Zahir · Área responsable: Registro de Internaciones — Hospitales Versión 1.0

📌 Objetivo

Automatizar el proceso de carga de registros de internación en CIPRES a partir de documentos generados por Integrando, procesados mediante bots de descarga, OCR y carga.

📝 Paso a Paso General

  1. El usuario solicita a través del sistema Integrando los listados de egresos mensuales, seleccionando el efector y el rango de fechas de egreso.
  2. Ejecutar la interfaz ubicada en app/interfaz.exe.
  3. En la interfaz, seleccionar únicamente los scripts:
    • descarga.py
    • ocr.py
  4. Los bots dejarán los archivos en:
    • inputs/pdfs/... — PDFs descargados.
    • inputs/excel_carga_cipres.xlsx — Excel generado por OCR.
  5. Enviar el Excel generado al usuario para su revisión manual.
  6. Una vez corregido, reemplazarlo en la carpeta inputs con el mismo nombre.
  7. Ejecutar nuevamente interfaz.exe, seleccionar solo carga_cipres.py y presionar INICIAR.

🧷 Configuración inicial

Desde la interfaz, presionar CONFIGURACIÓN y completar los siguientes campos:

CampoDescripción
Usuario / ContraseñaCredenciales de acceso a CIPRES
Ruta de la carpetaRuta absoluta al proyecto
Nombre ExcelDebe ser excel_carga_cipres.xlsx
Nombre Hoja ExcelEjemplo: Sheet1
URL_LOGIN / URL_INTERNURLs del sistema CIPRES
Usuario / Contraseña IntegrandoCredenciales de descarga
Fecha inicio / finRango del período
EfectorHospital o institución

Presionar Guardar Cambios.

▶️ Ejecución final

  1. Volver al menú principal.
  2. Seleccionar SELECCIONAR SCRIPT.
  3. Marcar solo carga_cipres.py.
  4. Presionar INICIAR.

📦 Resultados generados

  • outputs/cipres.xlsx: resultados por fila.
  • logs/evidencias/: logs, screenshots y errores.

📍Notas importantes

  • En caso de que un F10 esté incompleto, el usuario debe ingresar manualmente a la historia clínica para completar los datos.
  • Este paso no puede ser automatizado.
  • El Excel revisado debe tener todos los campos mínimos requeridos por CIPRES.
