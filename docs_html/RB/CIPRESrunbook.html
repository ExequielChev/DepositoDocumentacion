🧾 RUNBOOK - CARGA DE INTERNACIONES EN CIPRES

📌 Objetivo

Automatizar el proceso de carga de registros de internación en el sistema CIPRES, a partir de un archivo Excel procesado previamente con bots de descarga y OCR.

📁 Estructura del proyecto

Bots_CoParticipacion/
├── app/
│   ├── config/
│   │   └── settings.xlsx
│   ├── modulos/
│   │   ├── descarga.py
│   │   ├── ocr.py
│   │   └── carga_cipres.py
│   └── interfaz.exe
├── inputs/
│   ├── dic_diagnosticos_hombre.xlsx
│   ├── dic_diagnosticos_mujer.xlsx
│   ├── dic_causas_externas.xlsx
│   ├── excel_carga_cipres.xlsx
│   └── pdfs/
│       └── 2025-07-01-2025-07-31/
├── outputs/
│   ├── dataset/
│   └── cipres.xlsx
└── logs/
    └── evidencias/

🔧 Configuración Inicial

  1. Ejecutar interfaz.exe desde la carpeta app.
  2. Presionar CONFIGURACIÓN.
  3. Completar los campos:
Campo Descripción
UsuarioUsuario de acceso
ContraseñaContraseña
Ruta de la carpetaRuta completa al proyecto
Nombre ExcelRuta del archivo excel_carga_cipres.xlsx
Nombre Hoja ExcelGeneralmente Sheet1
URL_LOGINURL del login de CIPRES
URL_INTERNURL para carga de internaciones
EXCEL_OUTRuta para guardar resultados
Usuario_IntegrandoUsuario de Integrando
Contraseña_IntegrandoContraseña de Integrando
Fecha Inicio / FinRango de fechas de las internaciones
EfectorHospital o institución

Presionar Guardar Cambios al finalizar.

▶️ Ejecución

  1. Volver a la pantalla principal.
  2. Seleccionar SELECCIONAR SCRIPT.
  3. Marcar únicamente carga_cipres.py.
  4. Presionar INICIAR.
  5. Ver logs en tiempo real desde el visor en la interfaz.

📦 Resultados

  • outputs/cipres.xlsx: archivo con estado de cada internación.
  • logs/evidencias/: registros y capturas.

🛠️ Posibles errores comunes

Error Causa Solución
El bot no iniciasettings.xlsx mal configuradoRevisar rutas y credenciales
No encuentra archivo ExcelRuta o nombre de hoja incorrectosVerificar campos Nombre Excel y Hoja
Carga parcialDatos mal estructuradosRevisar salida del OCR y validación previa

📍Notas adicionales

  • La interfaz no requiere conexión para abrir, pero sí para acceder a CIPRES e Integrando.
  • Ejecutar solo cuando los datos estén validados correctamente.
